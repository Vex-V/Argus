# ============================================================================
# --- Optional heavy ML deps (facial + text analyzers) ---
# ============================================================================
# Large, slow to install, and some have brittle wheel availability on
# Python 3.13 / Windows. Kept below the core deps so `pip install -r
# requirements.txt` alone still gets you a working install.
#
#   pip install -r requirements.txt
#   pip install facenet-pytorch --no-deps   # see note below — separate step
#
# The services START and the pure-Python analyzers (timing, contacts, text
# stylometry) run WITHOUT these — the heavy imports are lazy. Install these to
# enable semantic text similarity, face matching, and language detection.
#
# Notes / py3.13 caveats:
#   * numpy stays at 2.x (installed above with the core deps).
#   * Pillow bumped to 12.x for full 3.13 support (see Analyzers section above).
#   * fastText (`fasttext`/`fasttext-wheel`) and InsightFace have no Python
#     3.13 wheels and fail to build from source on Windows without a C/C++
#     toolchain (onnxruntime/numpy pins). Replaced:
#       - language detection: fastText -> lingua-language-detector (ships a
#         cp313 wheel, bundles its own language models, no separate download).
#       - face embeddings: InsightFace -> facenet-pytorch (MTCNN detection +
#         InceptionResnetV1/VGGFace2 embedding, both 512-dim). facenet-pytorch
#         2.6.0's OWN pins (numpy<2, torch<2.3) are stale and would themselves
#         force a from-source numpy build on 3.13 — install torch/torchvision
#         normally first (modern cp313 wheels), THEN facenet-pytorch with
#         --no-deps so it doesn't downgrade them. The library's actual code
#         works fine against the newer stack; only its declared pins are old.
#
# First run downloads model weights: InceptionResnetV1/VGGFace2 (~110MB),
# paraphrase-multilingual-MiniLM (~120MB). Lingua's models are bundled in the
# wheel already (no download).

sentence-transformers==3.0.0
scipy==1.18.0
nltk==3.9.1
lingua-language-detector==2.2.0
torch==2.12.1
torchvision==0.27.1
facenet-pytorch==2.6.0  # install with --no-deps, see note above
